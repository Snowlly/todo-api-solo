# todo-api-solo

# Les Objectifs 
Créer une API REST de gestion de tâches, dockerisée avec MongoDB.

# Fonctionnalités
-  CRUD complet (Créer, Lire, Modifier, Supprimer)
-  Dockerisation complète (Dockerfile, docker-compose.yml)
-  Structure MVC simple
